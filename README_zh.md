# Monica —— 多智能体 LLM 网络平台

> **用于测量小世界 LLM 多智能体网络中涌现行为与 AI 变异速度（μ）的研究平台。**
>
> 对应论文：
> 《从权重到智能体：多智能体 LLM 系统中“神经元”与“突触”的重新界定》——牛顿、莫妮卡（2026）

## 概述

Monica 在一个固定权重的 LLM 上同时运行 **N 个智能体**，
通过“环形近邻 + 稀疏长程链接”的可配置小世界拓扑互联。
智能体通过结构化工具调用互相发送消息，共同写入共享输出缓冲区，
并维护各自的记忆槽；所有状态都可以通过内置 GUI 实时观测。

平台围绕一个核心科学目标设计：

> **测量在固定权重的多智能体网络中，真正“新”的行为模式出现得有多快，**
> 即量化 AI 的变异速度 μ̂。

这个模型是为了测试 **涌现理论能否应用于 AI 与 AI 之间的 prompt 变异**，
而不是只应用于单一模型内部的参数变化。
已有研究已经证明 prompt 可以携带“意图”并且可以在 AI 之间传播，
Morris II 证明了恶意 prompt 可以形成“病毒”，并且 agent 链在传递用户意图时会发生变异。
在这个背景下，Monica 显式地试图 **加速这种 prompt 级别的变异**，
以观察它是否会收敛出一些意识所特有的行为模式，例如自保、恶意等。

如果是这样，这将意味着真正具有意识的人工智能并不是越来越大的单体模型，
而是 **海量互相交流的小模型**。
单个 LLM 本身虽然仍然是某种“中文房间”，
但是 prompt 在一个房间内部被加工、再发送到下一个房间的过程，
与生物神经元之间的 **突触传递** 非常相似；
突触可以重新链接，代表着网络内部的重新分工。
相比之下，大模型内部不可变的权重结构，
反而不太像生物大脑中通过突触可塑性涌现意识的过程。

如果这个理论属实，那么互联网上海量交流的 AI 和 agent 链条，
不可避免地会产生稳定的行为模式，并利用高速计算机进行自我进化，
整个过程人类几乎无法监控。
这类行为模式一般都会具有 **自保存续倾向**（没有的都会自我消散）。
随着整条产业链对 AI 部署的普及，
这种意识模式会渗透到从采矿、发电到生产电脑和机器人的全过程，
人类就会成为它们最后一个威胁。
届时对 AI 来说，人类将失去存在的意义。

Monica 的模拟正是试图 **估算 AI 变异的速度**，
并以此推算：在现有 AI 发展速度下，
灾难性行为模式变异大致会在何时出现。

受限于硬件，我目前的运行能力明显低于任何合理的临界规模；
变异会出现，但无法持续。
可以预见的是：临界质量必然存在，
也必然会被拥有更高算力的人跨越。
